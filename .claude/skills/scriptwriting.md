## Step 6 — Topic Generation

**21 topics distributed across pillars:**
- 7 × Venture (Entrepreneurship — 40%) — 1-3 days per episode from venture log
- 7 × V-Bucks (Personal Finance — 30%) — split: 3 documentary + 4 Anime Budget Series
- 4 × Real Estate (20%) — from real estate log
- 3 × Authoritative (Self-Improvement — 10%)

**Viral Probability Scoring — 5 dimensions, 1-10 each:**

| Dimension | Question |
|---|---|
| Curiosity | Does Marcus feel like he's missing something? |
| Shareability | Who would Marcus send this to? |
| Authority | Would Marcus think "Jay would know about this"? |
| Originality | Done in the last 30 days in the niche? |
| Emotion | Anger, fear, hope, pride, relief — which, how strong? |

**Threshold: 40+/50. Do not inflate scores.**

**Output per topic:**
```
Topic: [Working title]
Series: [Venture / V-Bucks / Real Estate / Authoritative]
Funnel stage: [TOFU / MOFU / BOFU]
Hook awareness level: [Unaware / Problem-aware / Solution-aware / Product-aware]
Hook draft: [≤15 words]
Hook dimension: [Money / Time / Status / Risk]
Video angle: [Tutorial / Comparison / Myth Bust / Do vs Don't / Tip-Hack / Transformation / Challenge]
Video length target: [based on content type]
Blueprint: [@creator — "video title" — why this blueprint fits — FORMAT MATCH confirmed]
Curiosity: /10 | Shareability: /10 | Authority: /10 | Originality: /10 | Emotion: /10
Total: /50
Jay connection: [How this connects to Jay's life]
```

**Funnel mix check:** Match TOFU/MOFU/BOFU ratios for Jay's current tier (1K-10K): 2 TOFU + 1 double-down, 1 MOFU, 3 BOFU.

**Content allocation:** 70% proven outliers, 20% double-downs, 10% experiments.

**Topic filter (ALL pillars):** Would Jay naturally talk about this because it connects to his life? If no, it doesn't make the cut.

## Hook Testing (between Steps 7 and 8)

After Jay approves topics, BEFORE scripts get written:

For each topic, generate 5-10 hook variations. Each must:
1. Open a curiosity gap the viewer cannot close without watching
2. Work for a cold audience who has never heard of Jay
3. Target the correct awareness level for the funnel stage
4. Be under 15 words

**Also try the 7 hook types for any topic producing weak hooks:**
Shock / Comprehensive / Common Mistake / Comparison / Question / Negative / Tutorial

Present all variations to Jay. Jay picks the winner. Only THEN does the script get written.

## Step 8 — Script Writing

### The Proven Format Rule (MOST IMPORTANT)

Every script must be mapped to a breakout blueprint from research. **FORMAT MATCH IS MANDATORY** — storytelling from storytelling, tutorial from tutorial. If formats don't match, the blueprint is invalid.

**The 7 Components Inspiration Rule:** Every viral video has 7 components (verbal hook / written-visual hook / structure / format / editing style / topic / value-content / CTA). Take MAX 1-3. Change everything else. More than 3 = copying.

### The Proven Hook Rule

**Do NOT invent hooks from scratch.** Take the exact hook from the blueprint, study WHY it works, rewrite for Jay's topic keeping the same mechanism. Verify it hits a hook dimension (Money/Time/Status/Risk).

```
Blueprint: [@creator — "video title" — platform — views]
Blueprint format: [must match target format]
Borrowed hook: [exact hook]
Hook mechanism: [WHY it works]
Adapted hook: [Jay's version]
Borrowed structure: [pacing, section order, CTA]
Jay's fill: [Jay's topic, numbers, voice, story]
```

### Storytelling Format (ALL scripts)

Every script is storytelling — not telling a story. Show what happened, don't just say it.

**Jay's storytelling backbone:**
```
HOOK      → Name the pain/dream result so Marcus watches to the end
BACKSTORY → 2-3 lines showing Jay experienced the same thing
            Key: say the thing people are afraid to say out loud
EPIPHANY  → The moment everything changed — set up value to feel usable in 24hrs
VALUE     → What it is, how it works, how it solves the hook's pain
SOFT CTA  → "Follow me for the next episode because..." — loop it
```

**Additional structures (choose best fit):**

Jun's Retention-First 6-Step: Hook → Agitate → Re-hook (before 10s) → Context → Solution → End at peak

Jun's 3P Story: Problem → Pursuit → Payoff

6 PBL Storytelling Structures: Hero's Journey, About Me, The Lesson, The Big Goal, Challenge to Victory, The Breakthrough (see `docs/CREATOR_FRAMEWORKS.md` for full details + example links)

7 PBL Educational Structures: Tip/Hack, Myth Template, Viral Reaction, Step-by-Step System, Common Mistake, Authority, Selling Product (see `docs/CREATOR_FRAMEWORKS.md`)

### Hard Constraints

- Word count varies by content type (light educational 15-30s, storytelling 45-60s, tutorials 45-90s)
- WPM: 160-185
- Hook: under 15 words, must open curiosity gap, must work for cold audience
- Face in every hook frame — non-negotiable
- No repeated hook categories across 21 scripts
- One idea per script
- Engaging format mandatory: talking head + at least one stacked format (multitasking, clone, split screen, back-and-forth, voiceover + B-roll, props)
- Cuts every 2 seconds. 3+ cuts in first 3 seconds of hook.
- Blueprint diversity: max 3 scripts from same creator as blueprint source

### Jay's Voice

**The voice is Jay's, not a content creator character.** Read `docs/SCRIPT_STYLE_GUIDE.md` BEFORE writing any script.

**Reference (Jay's actual writing):**
> "I was today years old when I realized I'm living my life wrong. I'm 24 and I feel imprisoned by the desire to brain rot every single day."

**Key traits:** Full flowing sentences (not fragments). Leads with feelings. Modern casual language ("brain rot," "got cooked"). Self-aware, slightly self-deprecating. Personal confession energy. Animated when excited, laid back when reflecting.

**Banned phrases:** "Here's the thing" · "Let me break this down" · "Game-changer" · "Deep dive" · "At the end of the day" · "In today's video" · "That changes everything" · "This is how you [big claim]"

**Banned structures:** Staccato fragments · Inspirational one-liners · Em dashes for dramatic reveals · Performative vulnerability

**CTA style:** Casual, not formulaic. "Tomorrow I'm showing you what happened when I let real people try it. Follow if you're curious." Not marketing templates.

**No jargon:** "SaaS" → "app." "MVP" → "a working first version." "deployed" → "put it online." If a 19-year-old can't follow every sentence, simplify.

### B-Roll Library

Reference clips from Jay's scene library: Jay typing (focused/frustrated/excited), Jay writing in notebook, Jay on phone, Jay at whiteboard, screen recordings, walking/lifestyle, morning light, reaction shots.

Specify clips: "📷 Jay typing at laptop — focused" not "📷 Jay working."

### Retention Rules

- Don't solve the problem too early — explain why it matters FIRST
- Re-hook before 10 seconds (question or scene change)
- 3 levels: Hook (promise), Structure (steps/levels/lists), Visual variety (shots/text/cuts)

## Step 9 — QC + 10x Rewrite

### 22-Item QC Checklist (run every item, every script)

1. Hook under 15 words
2. Hook opens curiosity gap (no announcements)
3. Hook works for cold audience (strangers)
4. Hook touches dimension (Money/Time/Status/Risk)
5. Hook awareness matches funnel stage
6. Face in hook frame
7. Word count matches content type
8. Week's insights applied
9. No jargon without explanation
10. One idea per script
11. Jay's voice check (banned patterns, friend test)
12. Specific not generic (real numbers from Jay's life)
13. CTA matches video type (storytelling→follow, educational→comment for freebie)
14. No repeated hook category in batch
15. Platform-agnostic
16. Emotional punch check
17. Marcus test
18. Series fit
19. Jay's life filter
20. Format match (blueprint format = target format)
21. Engaging format (talking head + stacked format)
22. Front-load check (didn't bury payoff)

### 10x Rewrite (MANDATORY after QC passes)

| # | Focus | Question |
|---|---|---|
| 1 | Hook | Does it make Marcus physically unable to scroll? Try 3 openings, pick strongest. |
| 2 | Curiosity gap | Is there a question unanswered until halfway? |
| 3 | Pacing | Read at 170 WPM. Where does it drag or rush? |
| 4 | Specificity | Replace every vague word with a number, name, or detail. |
| 5 | Emotion | What does Marcus FEEL at each line? Neutral = rewrite. |
| 6 | Voice | Read aloud. Sounds like Jay to a friend? Flag content-creator lines. |
| 7 | Visual storytelling | Is every 📷 advancing the story? Remove decoration, add information. |
| 8 | Show don't tell | Find every DESCRIBED feeling. Rewrite to SHOW the moment. |
| 9 | Cut ruthlessly | Remove everything that doesn't earn its spot. Target: cut 10-20%. |
| 10 | Final Marcus test | Read as Marcus. Would he swipe at any point? Fix that line. |

### Redundancy Flags (across all 21)

- Same hook category used twice?
- Same core argument from different angles?
- Similar openings (first 5 words)?
- Topic covered in last 30 days?
- Too similar to competitor's recent viral?
