# Analytics Skill — Steps 11-12

**Purpose:** Review performance data, identify what worked and what failed, and feed learnings back into the pipeline. This skill runs Thursday-Friday.

---

## Step 11 — Analytics Review

**Timing:** 24-48 hours after posting. Don't check at 2 hours — the algorithm is still deciding.

### Metrics to Pull

| Metric | Platform | Why It Matters |
|---|---|---|
| Skip rate | IG | **Most critical.** Target: under 40%. Over 40% = hook failed. Over 60% = dead on arrival. |
| Views | Both | Raw reach |
| Watch time / Avg watch % | TikTok | Did they stay? |
| Average watch time (seconds) | IG | 4-5s = hook failure. 8s+ = hook working. |
| Shares | Both | Most important engagement metric |
| Saves | IG | High-intent engagement |
| Comments | Both | Emotional resonance |
| Follows from video | TikTok | Authority signal |
| Profile visits | Both | Curiosity signal |
| Accounts reached | Both | How wide the algorithm pushed it |

### What to Flag

- 🔥 **Breakout:** 3x the week's median views → add to breakout log → DOUBLE DOWN (reformat with different hook type or format)
- ❌ **Flop:** Under 0.5x median → note topic/hook for pattern analysis
- 📊 **Anomaly:** High shares + low views (good content, bad hook) OR high views + low follows (entertainment, not authority)

### Retention Curve Diagnosis

If a video flopped, check WHERE the drop-off happened:
- Drop at 3s = hook failed
- Drop at 10s = no re-hook / scene change
- Drop at 50%+ = solved problem too early or lost visual variety

### Jay's Weekly Analytics Feedback (MANDATORY)

Jay shares retention screenshots for every video. For each:
- Retention curve screenshot
- Skip rate
- Average watch time
- Accounts reached

Log this data to identify which hook types, structures, and formats perform best.

**Output:** Weekly analytics sheet at `content/analytics/[YYYY-WW]-analytics.md`. Top 3 and bottom 3 performers with hypotheses on why.

## Step 12 — Iterate

**Close the loop. Feed learnings back into Step 1.**

### Three Questions

1. **What topic category outperformed?** → Weight higher in next week's Steps 1 + 6
2. **What hook type got the most shares?** → Directional insight for next week's Step 8
3. **What flopped that we were confident about?** → Was it the topic, the hook, the script, or the delivery?

### Update Cycle

- Update `content/reference-creators.md` if a new creator is consistently producing breakouts
- Update voice rules if Jay's delivery consistently varies from what's written
- Update Key Rules if a pattern is strong enough to become standing
- Promote or demote formats in the framework roster based on Step 4E tracking
- Find outliers on Jay's page → reformat (same script, different format, re-film and re-post)

### Content Allocation Adjustment

Review the 70/20/10 split:
- 70% proven outliers — are the blueprints producing results?
- 20% double-downs — which of Jay's winners should be reformatted?
- 10% experiments — did any experiments earn a spot in the proven category?

### TOFU/MOFU/BOFU Ratio Check

Review funnel mix against Jay's current follower tier. As followers grow, ratios shift. Update the mix in the playbook when Jay crosses tier thresholds.

**The iterate principle:** Never run Week N+1 identical to Week N. Something always changes based on Week N's data.
