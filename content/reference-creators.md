**Format Tags:**
- **S** = Storytelling (voiceover + B-roll, personal narrative, showing what happened)

- **T** = Talking head (direct to camera, opinion, reaction)
- **D** = Documentary (day-in-the-life, build-in-public, process footage)
- **M** = Mixed/multiple formats

## Blueprint Matching Rules

1. **Format must match.** A storytelling script (S) must be blueprinted from a creator tagged S. An educational script (E) from a creator tagged E. A documentary script (D) from a creator tagged D.
2. **Series should match when possible.** A Venture script should first look to Venture-tagged creators. If no suitable blueprint exists there, check cross-series or general inspiration creators.
3. **If a creator has multiple format tags (e.g., S, T), only use videos that match the target format.** Don't pull a talking-head video from @ddennishan as a blueprint for a storytelling Venture script.
4. **If the curated list doesn't produce a suitable blueprint, search for additional creators in the same niche** following the 5x rule (10K-100K followers, views ≥ 5x follower count). Add strong new finds to this list.
