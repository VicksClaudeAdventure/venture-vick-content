# Proven Winners

This folder is currently empty. It will populate as scripts hit performance thresholds after being filmed and posted.

## Entry Criteria

A script moves into this folder when it hits AT LEAST ONE of the following after 48 hours of being live:

- Skip rate under 40%
- Views at 3x or more of the week's median (breakout)
- Shares or saves significantly above account baseline
- Follow conversion above account baseline

## Promotion Process

When a voice-reference script hits these thresholds:
1. Move the script file from `voice-reference/` to `proven-winners/`
2. Update the Performance Data section with:
   - Skip rate
   - Average watch time
   - Accounts reached
   - Likes, comments, shares, saves
   - Follow conversion
   - Retention curve screenshot (in assets/ subfolder if needed)
3. Add a "What Specifically Worked" section — 2-3 sentences of post-hoc diagnosis
4. Update the main `VOICE_DATABASE_INDEX.md` to reflect the promotion
5. Update `voice-markers.md` and `structural-patterns.md` if the winner reveals new patterns worth extracting

## Why This Matters

Proven-winners are weighted more heavily than voice-reference when Claude Code writes new scripts. A winning script proves that the voice AND structure worked together — not just that Jay's voice was present. Until we have proven winners, all structural choices should be cross-referenced against the broader pipeline's creator blueprints.
