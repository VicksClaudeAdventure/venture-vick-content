## Hook Mechanism Library (for Proven Hook Rule)

Every hook must be adapted from one of these proven mechanisms. Do NOT invent hooks from scratch.

### HOOK 1: Transparency Promise
**Source:** @chris.raroque — "Here's exactly how my app is doing financially three months after launch." — 249K
**Mechanism:** "Exactly" creates binding curiosity contract. Viewer expects REAL data.
**Used in:** V4, V6, VB1, VB3, VB4, RE4

### HOOK 2: Vulnerability Confession
**Source:** @chris.raroque — "I'm honestly embarrassed to admit this, but my app got hacked this week." — 150K
**Mechanism:** Shame + specificity. Lowering status RAISES trust. "This week" adds recency.
**Used in:** V5, VB6, RE2

### HOOK 3: Shocking Number + Question
**Source:** @humphreytalks — "$750 a month. Why is that the new average car payment in America now?" — 736K
**Mechanism:** Number stops scroll. Question keeps watching. Two-part hook.
**Used in:** V4, VB5

### HOOK 4: Unexpected Origin
**Source:** @ddennishan — "It all started when this girl broke my heart." — 241K
**Mechanism:** Emotional/unexpected event launching a business story. "Why does heartbreak lead to an app?"
**Used in:** V1, RE1, A1

### HOOK 5: Failure Count → Silver Bullet
**Source:** @jayhoovy — "I got rejected 47 times before I got my first investor." — 24K
**Mechanism:** Specific failure count + "one change" = irresistible "what changed?" curiosity.
**Used in:** V3

### HOOK 6: Extreme Effort Reveal
**Source:** @jayhoovy — "It took me 300 cold emails to land my first paying customer." — 5.5K
**Mechanism:** Large effort number creates awe. "One change" = must-know insight.
**Used in:** V2, A2

### HOOK 7: Mistake Confession
**Source:** @chris.raroque — "Number one mistake I made with my last app was not adding Google and Apple login on day one." — 102K
**Mechanism:** "Number one" implies ranking. Specific mistake + metric = practical value.
**Used in:** V7, VB7, A3

### HOOK 8: Moral Tension
**Source:** @raitryna — "It almost feels wrong making these kinds of videos, but..." — 49K
**Mechanism:** Breaking a social taboo (talking about money) creates psychological pull.
**Used in:** VB3

### HOOK 9: Authority Prescription
**Source:** @humphreytalks — "Every time you get paid, this is where your money should go." — 340K
**Mechanism:** Directive command from credible source creates compliance pull.
**Used in:** VB2

### HOOK 10: Date-Specific Event
**Source:** @humphreytalks — "On April 21st, these Vanguard ETFs will go from these prices to these prices." — 808K (HIGHEST breakout)
**Mechanism:** Urgency + before/after. "I need to know before the date."
**Used in:** RE3

### HOOK 11: Disbelief Milestone
**Source:** @raitryna — "I just did something I never thought would be possible because I just made $110,000." — 43K
**Mechanism:** "Never thought possible" opens belief gap. Specific amount is evidence.
**Used in:** VB2

### HOOK 12: Time-Compressed Transformation
**Source:** @raitryna — "I started posting on TikTok one year ago and since then I've made over a quarter million dollars." — 60K
**Mechanism:** Short time + large result = "how?" curiosity.
**Used in:** (available for future weeks)

## Blueprint Library (for Proven Format Rule)

### BP1: Revenue Transparency — @chris.raroque (249K)
**Structure:** number reveal → category breakdown → surprise → honest assessment → branded CTA
**Best for:** Venture (RentShield numbers), V-Bucks (spending/costs)

### BP2: Vulnerability Confession — @chris.raroque (150K)
**Structure:** confession → what happened → potential damage → how common it is → lesson → branded CTA
**Best for:** Venture (crises), any build-gone-wrong

### BP3: Emotional Origin — @ddennishan (241K)
**Structure:** emotional origin → what I built → it blew up → it broke → users validated → I use it myself
**Best for:** Venture (origin stories), Authoritative (personal arc)

### BP4: Building for Someone — @ddennishan (96K)
**Structure:** confession → backstory → failed attempt → creative pivot → audience invitation
**VOICE MODEL — closest to Jay's natural voice.**
**Best for:** Venture (personal motivation), humor-forward scripts

### BP5: Income Transparency — @raitryna (49K)
**Structure:** vulnerability disclaimer → goal update → category breakdown → gratitude → teaser CTA
**Best for:** V-Bucks (income, Recovery Series)

### BP6: Brand Deal Breakdown — @raitryna (60K)
**Structure:** milestone → anti-gatekeeping → numbered tips → personal timeline → permission CTA
**Best for:** V-Bucks (brand deals, creator economy)

### BP7: One Mistake Lesson — @chris.raroque (102K)
**Structure:** mistake → metric → someone's advice → fix → new metric → lesson → branded CTA
**Best for:** Venture (single-lesson beats), V-Bucks (money mistakes)

### BP8: Decision Walkthrough — @chris.raroque (136K)
**Structure:** credibility → category walkthrough → why each choice → proof → branded CTA
**Best for:** Venture (tool decisions), RE (decision process)

### BP9: Shocking Number Explainer — @humphreytalks (736K)
**Structure:** shocking number → explanation → practical context → takeaway
**Best for:** V-Bucks (cost reveals), Venture ($37.68)

### BP10: Date-Specific Opportunity — @humphreytalks (808K)
**Structure:** date → what changes → what to do → practical takeaway
**Best for:** RE (World Cup), any time-sensitive event

### BP11: Failure-to-Success — @jayhoovy (24K)
**Structure:** big failure count → one change → success → lesson
**Best for:** Venture (84% → 8%), any before/after beat

## Pattern Summary

**From 15 transcribed breakouts:**
- 8/15 hooks lead with a specific number in first 5-8 words
- 15/15 hooks are full flowing sentences (zero staccato)
- 6/15 open with transparency/confession
- Dominant trigger: Curiosity Gap (used in 7/15)
- Underused trigger with Reddit signal: Anger/Injustice (0/15 breakouts, massive Reddit appetite)
- Dominant structure: Story → Lesson (used in 6/15)
- @chris.raroque uses identical branded CTA across all 4 analyzed videos
